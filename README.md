## Formatowanie tekstu

Tekst może być formatowany na różne sposoby:

- **Pogrubienie** → `**Pogrubienie**`
- *Kursywa* → `*Kursywa*`
- ~~Przekreślenie~~ → `~~Przekreślenie~~`
- **_Pogrubiona kursywa_** → `**_Pogrubiona kursywa_**`
- `Kod w linii` → `` `Kod w linii` ``
